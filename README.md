<details>
<summary style="font-size: 18px; font-weight: bold; color: #FF00FF;">ğŸ’» Option 2: Local Development Setup</summary>

1. **Clone the repository:**
```bash
git clone https://github.com/saladtosser/campusconnect.git
cd campusconnect
```

2. **Set up the backend:**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

# Configure the database
# Create a PostgreSQL database named 'campusconnect'

# Apply migrations
python manage.py migrate

# Create a superuser
python manage.py createsuperuser

# Run the development server
python manage.py runserver
```

3. **Set up the frontend:**
```bash
cd frontend
npm install
npm start
```

4. **Access the application:**
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000/api/v1
- Admin interface: http://localhost:8000/admin

</details>
